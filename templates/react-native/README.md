# {{PROJECT_NAME}}

React Native mobile application.

## Requirements

- Node.js 18+
- React Native CLI
- Xcode (for iOS) or Android Studio (for Android)

## Quick Start

```bash
# Install dependencies
npm install

# Install iOS pods (macOS only)
cd ios && pod install && cd ..

# Start Metro bundler
npm start

# Run on iOS
npm run ios

# Run on Android
npm run android
```
